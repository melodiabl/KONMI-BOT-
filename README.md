## Configuración
Para ejecutar el backend es obligatorio definir la variable de entorno `JWT_SECRET` con un valor seguro. El servidor no iniciará si esta variable no está presente.

Variables adicionales recomendadas:

- `OWNER_WHATSAPP_NUMBER`: número de WhatsApp del propietario (solo dígitos) para otorgar permisos de administrador.
- `SUBBOT_EVENT_TOKEN`: token compartido con los sub‑bots para autenticar sus eventos.
- `GEMINI_API_KEY`: clave para habilitar las funciones de IA de Gemini.
